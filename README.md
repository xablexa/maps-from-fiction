# maps-from-fiction
## Data Repository for Castles, Battlefields, and Continents: A Dataset of Maps from Literature

The final dataset download includes:
- 2622 identified maps
- csv file of map filenames
- csv file of MARC record data for each map-novel (with annotations for map fictionality and scale)
- finetuned EfficientNet models (b0, b7, and V2_L)
